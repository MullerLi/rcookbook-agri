7.5.0.1 例 2.1 兩樣本的t檢定(標準差未知)

利用螢光胺基酸檢測兩種組織的螢光亮度，計算螢光亮度是否有差異。 | 觀測值 | 神經組織 | 肌肉組織 | | —— | ——– | ——– | | 1 | 6625 | 3900 | | 2 | 6000 | 3500 | | 3 | 5450 | 3450 | | 4 | 5200 | 3200 | | 5 | 5175 | 2980 | | 6 | 4900 | 2800 | | 7 | 4750 | 2500 | | 8 | 4500 | 2400 | | 9 | 3985 | 2200 | | 10 | 900 | 1200 | | 11 | 450 | 1150 | | 12 | 2800 | 1130 |

  • 計算： 神經組織平均值為\(\mu_1\)、肌肉組織為\(\mu_2\) | 組織 | 平均值 | 標準差 | | —- | ——- | ——- | | 神經 | 1917.99 | 960.51 | | 肌肉 | 4227.92 | 2534.17 |

因為兩樣本的標準差差異太大，所以不能確定 \(\sigma_1^2 = \sigma_2^2\) 需計算ｔ值與pooled自由度

  • 虛無假說 \(H_0: \mu_1 =\mu_2\) \(H_1: \mu_1 >\mu_2\)

  • 計算t值 \(t_0 = \frac{\mu_1-\mu_2} { \sqrt {{\frac {S_1^2}{n_1}}+ \frac{S_2^2}{n_2}{} }}\) \(t_0 = \frac {2309.93} {\sqrt{\frac{1917.99^2}{12}+\frac{960.51^2}{12}}} =2.7354\)

  • 計算pool樣本的自由度v： \[v = \frac{(\frac{S_1^2}{n_1}+\frac{S_2^2}{n_2})^2}{\frac{(\frac{S_1^2}{n_1})^2}{n_1}+\frac{(\frac{S_2^2}{n_2})^2}{n_2}}\]

v = 16.1995

\(t_0 = 2.7354\) \(t_{0.025,17}=0.690\) \(t_{0.025,16}=0.689\)

\(\therefore\) reject \(H_0\)

7.5.2 例 2.3 變異數檢定

抽樣新、舊機臺的樣本，測得資料如下，檢定兩樣本的變異數是否有差異。 | 機臺 | n | S^2 | | —- | — | —- | | 1 | 12 | 14.5 | | 2 | 10 | 10.8 |

  • 虛無假說 \(H_0: \sigma_0^2=\sigma_1^2\) \(H_1: \sigma_0^2 \ne \sigma_1^2\)

  • 檢定統計量 \(F_0=\frac{S_1^2}{S_2^2}=14.5/10.8=1.34\)

  • 棄卻域 \(F_0=1.34\) \(F_{0.05,12-1,10-1}=F_{0.05,11,9}=3.1\)

\(\because F_0 < 3.1\) \(\therefore DO NOT reject H_0\)

新舊機臺的變異數相等。

\(C=y_{..}^2/N=12355^2/20=7632301.25\) \(SS_t= \frac{1}{5}\sum_{i=1}^4y_{i.}^2-C=(2756^2+2937^2+3127^2+3535^2)/5-C=66870.55\) \(SS_T=\sum_{i=1}^a\sum_{j=1}^5{y_{ij}^2}-C=(575^2+...)-C=72209.75\) \(SS_E=72209.75-66870.55=5339.2\)

\(\because F_0=66.797 > F_{0.01,3,16}=5.29\) \(\therefore\) reject \(H_0\) at \(\alpha=0.001\)
